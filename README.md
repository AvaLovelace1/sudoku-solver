# sudoku-solver
Simple backtracking algorithm to solve a Sudoku puzzle.

## Input
A Sudoku puzzle with newlines after every line and no spaces. A zero denotes a missing number.

Example:
```
090018000
065002100
000000009
003091080
200000006
040360200
500000000
002500640
000830020
```

## Output
The completed Sudoku in the same format.

Example:
```
794618352
365942178
821753469
653291784
217485936
948367215
536124897
182579643
479836521
```
